## Проект построение модели определения стоимости автомобиля
Сервис по продаже автомобилей с пробегом  разрабатывает приложение для привлечения новых клиентов. В нём можно быстро узнать рыночную стоимость своего автомобиля. На основе исторические данные необходимо построить модель для определения стоимости автомобиля.

## Навыки и инструменты
- **pandas**
- **numpy**
- **matplotlib.pyplot**
- sklearn.model_selection.**GridSearchCV**
- **LightGBM**
- **CatBoostRegressor**

## Задача

Построить модель обученную определять рыночную стоимость автомобиля. Заказчику важны качество предсказания, скорость предсказания и время обучения модели. 

## Описание хода решения

В ходе решение проведены следующие этапы работ:
1. Загружен датасет, ликвидированы пропуски в данных
2. Прозведен анализ признаков, выведены определенные закономерности и наблюдения в данных
3. Обучены модели RandomForest, CatBoost, LightGBM.
4. На тест взята модель LightGBM, как показавшая наименьшее затрачиваемое на обучение время и соответствующее заявленнному значению необходимой метрики.

## Вывод

Построена модель, позволяющая предсказывать стоимость автомобиля по предложенным характеристикам. Выбрана наиболее "быстрая" модель.
Дальнейшие дествия необходимо согласовывать с заказчиком.
